## Introduction
为了更直观的理解 MapReduce 的概念和执行流程,根据 Google MapReduce 和 Apache MapReduce 自己实现单机和分布式 (on yarn) MR 框架

```
mapreduce-standalone/

            ├── mapper      map interface
            ├── reducer     reduce interface
            └── schedule    main process
```
